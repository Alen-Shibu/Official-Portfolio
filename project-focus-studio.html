The Details

Elegant Gallery Layout

Photography-first grid design that lets the images breathe. Carefully considered spacing and aspect ratios ensure every photo is displayed as intended.

02

Smooth CSS Animations

Scroll-triggered reveals, hover transitions, and page entrance animations built entirely with CSS — no JavaScript animation libraries needed.

03

Fully Responsive

Adapts gracefully from wide desktop displays down to mobile viewports. The gallery reflows naturally without breaking the visual hierarchy.

04

Zero Dependencies

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no build steps, no npm — just clean code that loads instantly and works everywhere.

Craft notes

Intentional Choices

No Framework

A deliberate decision to write everything from scratch — reinforcing understanding of how layout, animation, and interaction work at the browser level before reaching for abstractions.

Photography-First Design

Every design decision — spacing, color, typography — was made to support the photography, not compete with it. Dark palette, restrained UI, generous whitespace.

Performance by Default

CSS-only animations, optimised images, minimal JavaScript. The site loads fast because nothing unnecessary was added — a habit that carries into all my MERN projects.

Attention to Micro-interactions

Hover states, cursor changes, smooth transitions on every interactive element. These small touches are what separate a functional site from a polished one.
