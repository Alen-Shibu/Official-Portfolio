How it was built

My Approach

The backend was built first — a clean Express API with separate Admin and Location models, authMiddleware for route protection, and HTTP-only cookie sessions for security. The seed script means any new environment is one command away from a working state.

The frontend was then rebuilt from scratch in React with CSS Modules for true component-level style isolation. No global class leakage, no naming conflicts — just clean, composable components.

The Leaflet.js map with district accordion was a particularly satisfying piece — it gives the company a live, visual representation of their advertising footprint across Kerala.
